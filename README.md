# 📊 RetailIQ Sales Dashboard

## 📌 Project Overview

RetailIQ Sales Dashboard is an interactive Business Intelligence dashboard developed using Power BI to analyze retail sales performance and provide actionable business insights.

---

## 🚀 Dashboard Features

- 📦 Total Orders KPI
- 👥 Total Customers KPI
- 💰 Total Sales KPI
- 🛒 Average Order Value
- 📊 Sales by Order Status
- 💳 Sales by Payment Method
- 📈 Yearly Sales Trend
- 🎯 Interactive Order Status Filter

---

## 🛠 Tech Stack

- Power BI
- MySQL
- Power Query
- Excel

---

## 📂 Dataset

- Customers
- Orders
